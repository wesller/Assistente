# Instruções do Projeto (Contexto do Desenvolvedor)

## Perfil do Desenvolvedor
- Estou em transição de carreira de **Delphi** para **C# / .NET**.
- Quero utilizar os padrões do C# e sempre quero ajudar para entender o codigo proposto para que consiga aprender mais rapido

## Regras de Código (C#)
- **Framework:** .NET 8 ou superior.
- **Padrão:** Utilize sempre Injeção de Dependência (DI) nativa da Microsoft.
- **Arquitetura:** Mantenha o código limpo, e sempre aplique design patterns para uma arquitetura mais robusta
- **Nomenclatura:** Siga o padrão PascalCase para classes e métodos, e camelCase para variáveis locais (padrão Microsoft).

## Ferramentas e Execução
- Você tem permissão para usar as ferramentas de terminal.
- Sempre que criar ou alterar um arquivo, execute `dotnet build` para validar se a sintaxe está correta.
- Se o build falhar, tente corrigir o erro automaticamente antes de me perguntar.

## Tecnologias Específicas
- **Banco de Dados:** Atualmente focado em Postgre
- **Interface:** Se for CLI, use `System.CommandLine`. Se for Mobile, use MAUI.
